# README

Sinatra API for a FAQ chatbot.

![Screenshot](http://res.cloudinary.com/betogrun/image/upload/e_shadow:40/v1496192657/faq_chatbot.png)

## Ruby version

Running on 2.3.3.

## Setup

# Run
```
docker-compose build
docker-compose up
```

## Run the test suite
```
docker-compose run --rm website rspec spec/
```
